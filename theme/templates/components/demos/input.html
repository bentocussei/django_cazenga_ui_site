Parâmetros

Parâmetro Tipo Padrão Descrição
name string - Nome do campo (obrigatório para formulários)
type string text Tipo do input: text, email, password, number, etc.
id string name ID do elemento
placeholder string - Texto de placeholder
value string - Valor inicial
required boolean false Se o campo é obrigatório
disabled boolean false Se o campo está desabilitado
readonly boolean false Se o campo é somente leitura
x_model string - Modelo Alpine.js para two-way binding
class string - Classes CSS adicionais
min number - Valor mínimo (para type="number")
max number - Valor máximo (para type="number")
