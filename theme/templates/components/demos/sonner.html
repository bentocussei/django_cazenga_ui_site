Exemplos de Código

Toast Básico

showToast('success', 'Sucesso!', 'Operação concluída com sucesso.')

Toast com Ação

showToast('info', 'Nova Mensagem', 'Você tem uma nova mensagem.', {
  action: {
    text: 'Visualizar',
    handler: () => window.open('/messages')
  }
})

Toast Permanente

showToast('warning', 'Atenção', 'Ação requerida', {
  duration: 0,
  closable: true
})

Toast Customizado

showToast('error', 'Erro Crítico', 'Falha no sistema', {
  duration: 10000,
  position: 'top-center',
  closable: false
