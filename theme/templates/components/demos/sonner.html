Instalação

O componente Sonner deve ser incluído no template base uma vez:

{% verbatim %}{% include "components/ui/sonner.html" %}{% endverbatim %}

Tipos de Notificação

Diferentes tipos de toast para diferentes situações.

showToast('Título', 'tipo', 'Descrição')

Toast com Ação

Toast que permite interação do usuário.

showToast('Título', 'tipo', 'Descrição', { action: { text: 'Botão', handler: () => {} } })

Toast Permanente

Toast que não desaparece automaticamente.

showToast('Título', 'tipo', 'Descrição', { duration: 0 })

Posições

Toast pode aparecer em diferentes posições na tela.

{% for example in sonner_positions.examples %} {% endfor %}
showToast('Título', 'tipo', 'Descrição', { position: 'top-right' })

Configurações Especiais

Toast com configurações avançadas.

showToast('Título', 'tipo', 'Descrição', { duration: 8000, closable: false })
